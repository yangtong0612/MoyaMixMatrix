# Subtitle sound effects

All sound effects in this folder are free CC0/public-domain assets used for subtitle-template previews.

## Sources

- `opening-pop-soft.ogg`, `opening-pop-light.ogg`: OpenGameArt Pop sounds, CC0.
  - https://opengameart.org/content/pop-sounds
- `opening-hit-light.ogg`: Kenney Impact Sounds, Creative Commons CC0.
  - https://www.kenney.nl/assets/impact-sounds
- `opening-whoosh-gentle.wav`, `transition-whoosh-fast.wav`, `transition-whoosh-short.wav`: CC0-Public-Domain-Sounds, Micro Pack - Organic Wooshes.
  - https://github.com/lavenderdotpet/CC0-Public-Domain-Sounds
- `transition-glitch-soft.ogg`, `emphasis-soft-pop.ogg`, `emphasis-bright-ding.ogg`, `emphasis-click-clack.ogg`: Kenney Interface Sounds, Creative Commons CC0.
  - https://www.kenney.nl/assets/interface-sounds
- `transition-glitch-sci-fi.ogg`, `emphasis-tech-beep.ogg`: CC0-Public-Domain-Sounds, 50 CC0 Sci-Fi SFX.
  - https://github.com/lavenderdotpet/CC0-Public-Domain-Sounds
