# 彤见矩阵产品说明书 v1.3（重建版）

定位：AI 短视频增长交付操作系统  
核心口号：让每个卖点都被看见

## 产品定位

彤见矩阵是一套支撑培训转化、代理交付、企业内训、终端案例复盘的 AI 短视频增长交付操作系统。

它把原本依赖人经验完成的短视频服务，拆成可训练、可授权、可执行、可质检、可复盘的数据化流程。

## 四类使用者

| 使用者 | 使用目的 |
| --- | --- |
| 平台总部 | 管课程、代理、模板、质检、数据、结算 |
| 认证代理 | 获客、建档、生成、交付、复盘 |
| 企业客户 | 训练内部团队，建立短视频 SOP |
| 终端商家 | 提交素材，获得内容测试包和复盘 |

## 产品架构

1. 培训转化层
2. 代理交付层
3. 内容生产层
4. 数据质检层

## 核心工作流

### 商家内容测试工作流

素材上传 -> 卖点提炼 -> 多角度脚本 -> 内容测试包 -> 质检 -> 发布 -> 数据回收 -> 模板优化

### 代理交付工作流

学员报名 -> 代理认证 -> 客户录入 -> 生成任务 -> 内容交付 -> 复盘 -> 评级结算

### 企业培训工作流

企业内训 -> 员工作业 -> 专属模板 -> 系统升级 -> 年度顾问

## MVP

- 线索和学员管理
- 代理客户档案
- 素材采集表
- 餐饮团购利益点提炼
- 内容测试包生成
- 质检状态
- 发布台账
- 复盘数据录入
- 基础结算记录
