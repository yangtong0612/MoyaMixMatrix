# 彤见矩阵产品说明书 v1.0（重建版）

定位：AI 商业短视频内容工厂  
核心口号：让每个卖点都被看见

## 产品定位

彤见矩阵是一套面向本地生活商家、电商商家和短视频运营团队的 AI 商业短视频内容工厂。

产品目标不是替代传统剪辑软件，而是帮助用户把商品图、门店图、详情页、活动信息、爆款参考和核心卖点，快速转化为可发布、可测试、可复用的短视频内容。

## 目标人群

- 电商商家：商品图、详情页、卖点转带货视频。
- 本地生活商家：门店图、活动海报、团购信息转同城引流视频。
- 短视频矩阵团队：批量生成多版本视频做账号矩阵发布。
- MCN / 代运营：提高内容交付效率。
- 达人 / 主播 / 个人 IP：口播、直播切片、课程推广。

## 核心痛点

- 不会写脚本。
- 不会表达产品利益点。
- 不会剪辑。
- 内容产量不够。
- 爆款难复刻。
- 素材分散，无法沉淀。

## 核心功能

1. 商品视频生成
2. 门店引流视频生成
3. 爆款结构参考
4. 网感剪辑
5. 极速裂变混剪
6. 成片库
7. 矩阵发布

## 产品价值

传统剪辑工具解决“怎么剪”，彤见矩阵解决“该表达什么、如何批量表达、如何形成持续内容产能”。

## 阶段说明

v1.0 是最早的产品构想版，范围较宽，覆盖电商、本地生活、达人、MCN 和代运营等多个群体。后续版本逐步收窄到更明确的第一切口。
